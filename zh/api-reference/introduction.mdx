## 基础 URL

所有 API 请求都发送到：

```
https://www.anyfast.ai
```

## 身份认证

AnyFast API 使用 Bearer Token 进行认证。在每个请求的 `Authorization` 请求头中携带你的 API Key：

```bash
Authorization: Bearer YOUR_API_KEY
```

你可以在 [控制台](https://dashboard.anyfast.ai) 中创建和管理 API Key。

## 请求格式

所有请求体均为 JSON 格式，请设置对应的 `Content-Type` 请求头：

```
Content-Type: application/json
```

## 错误码

| HTTP 状态码 | Code | 说明 |
|---|---|---|
| 400 | `invalid_request` | 参数缺失或无效 |
| 401 | `unauthorized` | API Key 无效或缺失 |
| 429 | `rate_limit_exceeded` | 请求过于频繁 |
| 500 | `internal_error` | 服务器内部错误 |

所有错误响应格式如下：

```json
{
  "error": {
    "code": "invalid_request",
    "message": "字段 'prompt' 不能为空"
  }
}
```
