---

Upmore 完全兼容 Anthropic Messages API。使用官方 Anthropic SDK，只需更改 base URL 即可。

## 端点

```
POST https://api.upmore.net/v1/messages
```

## 认证

将你的 Upmore API 密钥作为 Bearer token 传入：

```
Authorization: Bearer YOUR_API_KEY
```

## 快速示例

<CodeGroup>

```bash cURL
curl https://api.upmore.net/v1/messages \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      { "role": "user", "content": "你好，Claude！" }
    ]
  }'
```

```python Python
import anthropic

client = anthropic.Anthropic(
    api_key="YOUR_API_KEY",
    base_url="https://api.upmore.net"
)

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "你好，Claude！"}
    ]
)
print(message.content[0].text)
```

```javascript Node.js
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: "YOUR_API_KEY",
  baseURL: "https://api.upmore.net",
});

const message = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [{ role: "user", content: "你好，Claude！" }],
});
console.log(message.content[0].text);
```

</CodeGroup>

## 支持的模型

| 模型 | 描述 |
|------|------|
| claude-sonnet-4-6 | 最新最强的 Sonnet 模型 |
| claude-sonnet-4-5-20250929 | Claude Sonnet 4.5 |
| claude-sonnet-4-20250514 | Claude Sonnet 4 |
| claude-opus-4-6 | 最新最强的 Opus 模型 |
| claude-opus-4-5-20251101 | Claude Opus 4.5 |
| claude-opus-4-1-20250805 | Claude Opus 4.1 |
| claude-opus-4-20250514 | Claude Opus 4 |
| claude-haiku-4-5-20251001 | 快速且经济的 Haiku 模型 |
| claude-3-7-sonnet-20250219 | Claude 3.7 Sonnet |
| claude-3-5-sonnet-20241022 | Claude 3.5 Sonnet |

## 功能特性

- **扩展思考** — 通过 `thinking` 参数启用深度推理
- **流式输出** — 设置 `stream: true` 通过 SSE 实时流式传输
- **工具调用** — 通过 `tools` 定义工具，通过 `tool_choice` 控制使用方式
- **视觉理解** — 通过 `type: "image"` 内容块发送图片
- **系统提示** — 使用 `system` 参数提供上下文和指令

## 扩展思考示例

```python
message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=16000,
    thinking={
        "type": "enabled",
        "budget_tokens": 10000
    },
    messages=[
        {"role": "user", "content": "请一步步解答：27^(1/3) * 16^(1/4) 等于多少？"}
    ]
)
```
