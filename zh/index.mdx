<Note>
  **仅限内部使用。** Upmore 是面向公司内部员工的内部服务，**不对外提供服务**，也**不提供注册**——仅企业内部员工通过企业单点登录（SSO）授权访问。
</Note>

Upmore 是内部 AI API 网关，让公司内部员工通过单一、OpenAI 兼容的接口访问组织已审批的 AI 模型——涵盖对话、图片生成和视频生成。

无需为每家提供商单独管理密钥、SDK 和认证流程，Upmore 将一切统一到一个端点。只需修改一个参数，即可在 ByteDance、OpenAI 和 MiniMax 的模型之间切换。

## Upmore 提供什么

**通过企业单点登录（SSO）访问**
不提供公开注册。员工使用企业 SSO 账号登录后，为自己的应用创建 API Key。

**统一的 OpenAI 兼容端点**
对话、图片、视频模型全部通过一个 Base URL 和一个 Key 访问。已在用 OpenAI SDK 的话，只需改 Base URL。

**多模态支持**
除对话外，Upmore 以清晰的格式暴露图片和视频生成，及各提供商特有的参数。

**一致的错误处理**
每个端点返回统一的错误码和消息格式，你的错误处理逻辑保持简单。

**内部成本核算**
用量按 token（对话）和按次（媒体）计量，用于内部成本归集。
