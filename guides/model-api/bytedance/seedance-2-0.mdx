Seedance 2.0 is ByteDance's video generation model, capable of producing high-quality videos from text descriptions. It supports multiple resolutions and durations, making it suitable for content creation, prototyping, and creative workflows.

## Key capabilities

- **Text-to-video** — Generate videos directly from natural language prompts
- **Resolutions** — Supports `720p` and `1080p` output
- **Duration** — Choose between 5-second and 10-second clips
- **Async generation** — Submit a task and poll for the result when ready

## Quick example

<CodeGroup>

```bash cURL
# Step 1: Submit generation task
curl https://www.anyfast.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "prompt": "A serene mountain lake at sunrise, cinematic 4k",
    "duration": 5,
    "resolution": "1080p"
  }'

# Step 2: Poll for result
curl https://www.anyfast.ai/v1/videos/generations/task_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import time
import requests

headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

# Step 1: Submit task
response = requests.post(
    "https://www.anyfast.ai/v1/videos/generations",
    headers=headers,
    json={
        "model": "seedance-2.0",
        "prompt": "A serene mountain lake at sunrise, cinematic 4k",
        "duration": 5,
        "resolution": "1080p"
    }
)
task_id = response.json()["id"]

# Step 2: Poll until done
while True:
    result = requests.get(
        f"https://www.anyfast.ai/v1/videos/generations/{task_id}",
        headers=headers
    ).json()

    if result["status"] == "completed":
        print(result["video_url"])
        break
    elif result["status"] == "failed":
        raise Exception(result["error"])

    time.sleep(3)
```

</CodeGroup>

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `model` | string | Yes | Must be `seedance-2.0` |
| `prompt` | string | Yes | Text description of the video |
| `duration` | integer | No | `5` or `10` seconds. Default: `5` |
| `resolution` | string | No | `720p` or `1080p`. Default: `720p` |
| `seed` | integer | No | Fixed seed for reproducible results |
