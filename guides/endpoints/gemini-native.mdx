---

Upmore is compatible with the Google Gemini native API. Use the official Google Generative AI SDK — just change the endpoint to Upmore.

## Endpoint

```
POST https://api.upmore.net/v1beta/models/{model}:generateContent
```

## Authentication

Include your Upmore API key as a Bearer token:

```
Authorization: Bearer YOUR_API_KEY
```

## Quick example

<CodeGroup>

```bash cURL
curl https://api.upmore.net/v1beta/models/gemini-2.5-pro:generateContent \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [{ "text": "Hello, Gemini!" }]
      }
    ]
  }'
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="YOUR_API_KEY",
    transport="rest",
    client_options={"api_endpoint": "https://api.upmore.net"}
)

model = genai.GenerativeModel("gemini-2.5-pro")
response = model.generate_content("Hello, Gemini!")
print(response.text)
```

```javascript Node.js
import { GoogleGenerativeAI } from "@google/generative-ai";

const genAI = new GoogleGenerativeAI("YOUR_API_KEY");
// Configure to use Upmore endpoint
const model = genAI.getGenerativeModel(
  { model: "gemini-2.5-pro" },
  { baseUrl: "https://api.upmore.net" }
);

const result = await model.generateContent("Hello, Gemini!");
console.log(result.response.text());
```

</CodeGroup>

## Features

- **System instructions** — Use `systemInstruction` for context and instructions
- **Function calling** — Define functions via `tools.functionDeclarations`
- **Structured output** — Set `generationConfig.responseMimeType` to `"application/json"` with a `responseSchema`
- **Safety settings** — Configure content filtering with `safetySettings`
- **Multi-modal input** — Send images via `inlineData` in content parts
