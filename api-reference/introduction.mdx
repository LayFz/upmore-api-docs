## Base URL

All API requests should be made to:

```
https://www.anyfast.ai
```

## Authentication

The AnyFast API uses Bearer tokens for authentication. Include your API key in the `Authorization` header of every request.

```bash
Authorization: Bearer YOUR_API_KEY
```

You can create and manage API keys in the [Dashboard](https://dashboard.anyfast.ai).

## Request Format

All request bodies must be JSON. Set the `Content-Type` header accordingly:

```
Content-Type: application/json
```

## Error Codes

| HTTP Status | Code | Description |
|---|---|---|
| 400 | `invalid_request` | Missing or invalid parameters |
| 401 | `unauthorized` | Invalid or missing API key |
| 429 | `rate_limit_exceeded` | Too many requests |
| 500 | `internal_error` | Server error |

All errors follow this format:

```json
{
  "error": {
    "code": "invalid_request",
    "message": "Field 'prompt' is required"
  }
}
```
